## Overview
Money Manager is a **full-stack web application** designed to help users efficiently manage their income, expenses, and categories. It offers secure authentication, real-time transaction tracking, powerful data filtering, and visual dashboards‚Äîbuilt with Java, Spring Boot, MySQL, React, and JWT for reliability and a seamless user experience.
